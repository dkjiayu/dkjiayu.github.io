C语言之指针

2018-12-09 C语言

#声明：星号 在此处显示✳
指针是一个变量，指向一个地址，指向数据的类型与定义时有关

基类型✳名称
int✳p; 代表指针变量p指向的将会是一个整型数据—指针就是一个地址
当然指针变量本身也有一个地址，这将涉及到后面的二级指针，指针数组
注意：定义指针变量时 ✳号是固定格式。离开定义指针变量时 ✳号就代表取地址运算符

指针与数组

一维数组名称默认代表着第一个元素的地址，也就是该一维数组的起始地址。
同时要注意传值与传地址的区别：定义一个函数：传值不能直接对根本数据修改，而传地址是直接对根本数据进行修改。

二维数组与地址的关系又复杂了一点。

二维数组可以看成“一个一维数组”。每一行代表一个元素，于是出现了行地址与列地址
由一维数组可知数组名称代表首元素也即该数组的起始地址。二维数组的名称也就是“该一维数组的首元素的地址”，该特殊一维数组的首元素也就是第一行，所以二维数组的名称代表首行地址。
假如一个二维数组为a[3][3]，那么该二维数组可以看成一个二维数组a[3]={a[0],a[1],a[2]}而a[0]={a[0][0],a[0][1],a[0][2]},其他也一样。所以a代表a[0]的地址，虽然a和a[0]地址相同但是+1后意义不一样。
a[0]代表着行地址，a[0][0]代表着列地址。a+1 代表着行数加1 相当于a[1]; 而a[0]+1 代表着列数加1，相当于a[0][1]的地址

理清二维数组是个什么东东之后再看指针

#利用数组名读取

行指针与列指针的互相转换。
先看一个例子：int a=5,✳b=&a; 指针b指向a的地址， ✳b即a中的内容5， &a即取a的地址，就会得到指针b。
这相当于一个来往。✳b得到a,&a又可以得到b；
同样在二级指针这里，a指向a[0],✳a得到a[0],&a[0]又得到a。这样就出现了行列地址的互相转换。
所以有✳(a+i)=a[i] a+i 代表行地址（i每+1就会增加一行） a[i]代表列地址（i每+1就会移动一列）
若要利用指针读取二维数组元素，首先对数组进行定位，由行切进列，就得到一个元素的地址。例如a[m][n],
先定义到行，可以a+m,进入列首✳(a+m)，进一步确定✳(a+m)+n，然后取值✳（✳(a+m)+n)
当然简单的就是✳(a[m]+n)

#利用一级指针读取

所谓利用一级指针，就是把指针精确到元素，然后进行遍历全数组
例如int✳b=&a[0][0] ,b+1则指向a[0][1], a[0][1]==✳(b+1)
由此也联系到下标法，a[i✳m+n] 其中i是第几行，m是一行的元素个数，n是第n列。由此容易理解下标法是利用一级指针逐个元素进行读取的。

#利用行指针进行读取

这个需要注意的就是行指针变量的定义，用法也就是用法一。只不过找个指针变量来代替二维数组名字
int (✳b)[3] 类型标识符 (✳指针变量名) [二维数组的列数]
定义行指针变量的时候之所以用括号是为了和 下面指针数组进行区分。

#指针数组

先来定义一波： 类型标识符✳指针变量名[数组的长度]
首先要明白指针数组元素的是指针
对于定义格式的简单理解：可以和指针的定义相结合理解，✳号在此处声明这是个指针数组。

#二级指针

定义： 类型标识符✳✳指针变量名
指向指针的指针
int a=5,✳b=&a,✳✳c=&b;
b指向a,存储的是a的地址
c指向指针b，存储的是指针b的地址
✳c 得到b，✳✳c得到a。
